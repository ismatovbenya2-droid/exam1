# README

В данной работе я сверстал сайт по макету Figma «Кабинет метролога». Этот сайт предназначен для метрологических организаций и лабораторий. На сайте представлены возможности управления оборудованием, документацией, сотрудниками и сроками выполнения работ.

В процессе выполнения проекта я закрепил навыки работы с HTML и CSS, научился использовать Flexbox, Grid, работать с изображениями и создавать адаптивную верстку для планшетов и мобильных устройств.

В результате получился многостраничный адаптивный интерфейс, который соответствует предоставленному макету Figma.
